Question-1: What are the different ways to select an element in the DOM?

There are several ways to select elements in the DOM using JavaScript. The most common method is getElementById(), which selects a single element by its ID. You can also use getElementsByClassName() or getElementsByTagName() to retrieve elements by class or tag name. For more flexibility, querySelector() allows selecting the first element that matches any CSS selector, while querySelectorAll() returns all matching elements as a NodeList. These methods are often combined with DOM traversal techniques to target specific elements.

Question-2: What is the difference between innerHTML, innerText, and textContent?

The key difference between innerHTML, innerText, and textContent lies in how they handle content and formatting. innerHTML returns the HTML structure as a string, including tags. innerText only returns visible text, ignoring hidden elements and respecting CSS styles. textContent returns all the text within an element, regardless of styling or visibility. While innerHTML is useful for injecting HTML, innerText and textContent are better suited for retrieving plain text.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in JavaScript where a parent element handles events for its child elements. Instead of attaching individual event listeners to each child, you add one listener to the parent and use conditional logic to detect which child triggered the event. This approach improves performance and makes your code more efficient, especially when working with dynamically added elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a behavior in the DOM where an event triggered on a child element propagates upward through its parent elements. For example, if a click occurs on a button inside a div, the event will "bubble up" to the div and any other ancestors. This allows parent elements to listen for events that happen on their children without needing separate event listeners for each one.

Question-5: How do you create, add, and remove elements using JavaScript?

To create a new element in JavaScript, you use document.createElement("tagName"). After creating it, you can set its properties or inner content, and then use appendChild() or append() to add it to the DOM. To remove an element, you can call remove() on the element itself or use parent.removeChild(child). These methods give you full control to dynamically update the page structure with JavaScript.
