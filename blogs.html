Queston-1: What are the different ways to select an element in the DOM?

ChatGPT said: In JavaScript, DOM elements can be selected using various methods depending on the need. document.getElementById("id") selects a single element by its unique ID, while document.getElementsByClassName("class") and document.getElementsByTagName("tag") return collections of elements matching a class or tag name respectively. For form elements, document.getElementsByName("name") retrieves elements by their name attribute. The most flexible and modern approach is using document.querySelector("selector") to select the first element that matches a CSS selector, and document.querySelectorAll("selector") to select all matching elements, returning a NodeList. Additionally, elements can be selected by traversing from a parent element using methods like parent.querySelector(".child").
